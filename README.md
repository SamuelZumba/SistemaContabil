# SISTEMA DE CONTABILIDADE CONTABILIDADE - Sistema de Gestão e Lançamentos

Este é um sistema de gestão contábil completo, desenvolvido para organizar fluxos financeiros, cadastros de clientes e automação de relatórios contábeis. O projeto foi estruturado com foco em lógica de programação robusta utilizando JavaScript puro e design responsivo.

# Funcionalidades Principais

# Área do Usuário & Segurança
Controle de Acesso:Sistema de login e cadastro com suporte a perfis de Administrador e Colaborador.
Validação em Tempo Real:Verificação de CPF/CNPJ, força de senha e confirmação de dados durante o registro.
Persistência de Dados:Uso de LocalStorage para manter as informações de usuários, clientes e lançamentos salvas no navegador.

# Gestão Contábil
Cadastro de Clientes: Interface para gerenciar dados de clientes e empresas com busca filtrada.
Lançamentos Financeiros:Registro detalhado de contas, permitindo selecionar tipos contábeis como Ativo, Passivo, Receita e Despesa.
Histórico de Lançamentos:Visualização organizada com filtros por cliente e soma total de registros.

# Relatórios e Inteligência de Dados
DRE e Balanço:Geração automática de Demonstração do Resultado do Exercício e Balanço Patrimonial em formato de texto.
Gestão de Arquivos:Possibilidade de anexar documentos e importar arquivos diretamente para o perfil do cliente.
Exportação:Função para realizar download de relatórios e dados cadastrais no formato `.doc`.

# Tecnologias Utilizadas

* **Frontend:** HTML5 e CSS3 moderno com suporte a Modo Escuro (Dark Mode).
* **Framework CSS:** Bootstrap 5.3 para responsividade e componentes de interface.
* **Lógica de Programação:** JavaScript Vanilla (ES6+) para manipulação de DOM e lógica de negócios.
* **Ícones:** Font Awesome 6.5.
